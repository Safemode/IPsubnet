Résumé:

La plus grande taille allouable possible est /32, mais est réservé pour les providers internet.
/48 est une taille typique pour une grand une entreprise. Dans certains cas une taille plus grande peut être fournie mais avec une raison précise.
De plus au dessus /48 les IP doivent être declarées au RIR (exemple pour l'Europe au RIPE NCC) avec un plan d'implémentation.
Dans IPv6 le nombre d'appareils n'est pas important,' IP étant formée de l'ID de l'interface, donc le subnet standard est /64.
L'exception est pour les routeurs (/127). [https://tools.ietf.org/html/rfc6177]

Vocabulaires:


IANA = responsable pour la coordination globale l'internet.
RIR = Registre régional d'internet. RIPE NCC (Europe,Middle East,Central Asia), ARIN (Canada,USA), LACNIC (Latin America), AFRINIC (Africa), APNIC (Asia/Pacific)
LIR = Registre local pour internet. En fait le direct provider de votre compagnie.
PI = Provider Independent Assignments

Detail:

IPv6 assignation: IANA->RIR->LIR->client

Format:

Prefix FF00::/8. adresse IPng multicast. Identificateur pour un groupe d'interfaces.
A interface may belong to any number of multicast groups. Multicast addresses have the following format:
      |   8    |  4 |  4 |                  112 bits                   |
      +------ -+----+----+---------------------------------------------+
      |11111111|FLGS|SCOP|                  GROUP ID                   |
      +--------+----+----+---------------------------------------------+
      Prefix
       11111111 at the start of the address identifies the address as being a multicast address.
      FLGS
      It is a set of 4 flags: |0|0|0|T| +-+-+-+-+
        The high-order 3 flags are reserved, and must be initialized to 0.
        T=0 indicates a permanently assigned (well-known) multicast address, assigned by the global internet numbering authority.
        T=1 indicates a non-permanently assigned (transient) multicast address.
      SCOP
         SCOP is a 4-bit multicast scope value used to limit the scope of the multicast group. The values are:
      0 Reserved 8 Organization-local scope 1 Node-local scope 9 (unassigned) 2 Link-local scope A (unassigned)
      3 (unassigned) B (unassigned) 4 (unassigned) C (unassigned) 5 Site-local scope D (unassigned) 6 (unassigned) E Global scope
      7 (unassigned) F Reserved.
      GROUP
      GROUP ID identifies the multicast group, either permanent or transient, within the given scope.
